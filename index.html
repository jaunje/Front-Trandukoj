What is this?

Tradukoj is a collaborative platform that allows developers to concentrate on tasks and leave the translations to professio that allows developers to concentrate on tasks and leave the translations to professionals



How it works?

Bundle

First you need to register an account and create your project, then assign a key and secret and finally install in your project then bundle translations-apibundle.

In the bundle documentation have complete information on how to sync translations in your symfony2 project with translations interface.

What we offer?

Unlimited projects per user account

Unlimited users per project with role hierarchy

Unlimited keys per project

Unlimited documents per project
